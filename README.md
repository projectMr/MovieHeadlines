### 电影头条 webAPP单页（react）
#### 之前有一个快递查询的单页，这个电影头条也是其中的一个功能，用到了create-react-app作为脚手架构建,搭配Ant Design Mobile，不一样的地方是，虽然电影头条上面的信息比较少，但是上拉加载、下拉刷新、路由跳转都是存在的，而且接口是真实的。很适合学习。
